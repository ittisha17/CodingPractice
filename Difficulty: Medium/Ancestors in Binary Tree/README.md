<h2><a href="https://www.geeksforgeeks.org/problems/ancestors-in-binary-tree/1?page=4&category=Tree,Graph,Linked%20List,Divide%20and%20Conquer&company=Amazon&difficulty=Medium&sortBy=difficulty">Ancestors in Binary Tree</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18px;">Given a Binary Tree and an integer <strong>target, </strong>find all the ancestors of the given target.</span></p>
<p><span style="font-size: 18px;"><span style="font-size: 14pt;">Note:<strong> </strong></span></span></p>
<ul>
<li><span style="font-size: 18px;"><span style="font-size: 14pt;">A<span style="background-color: #ffffff; color: #040c28; font-family: 'Google Sans', arial, sans-serif;">&nbsp;node y is an ancesto x<strong> </strong>if at the upper level of <strong>x </strong>and<strong> </strong>on the path from x to root.&nbsp;</span></span></span></li>
<li><span style="font-size: 18px;"><span style="font-size: 14pt;"><span style="background-color: #ffffff; color: #040c28; font-family: 'Google Sans', arial, sans-serif;">Root is ancestor of all other nodes and there is no ancestor of the root.</span></span></span></li>
<li><span style="font-size: 18px;"><span style="font-size: 14pt;"><span style="background-color: #ffffff; color: #040c28; font-family: 'Google Sans', arial, sans-serif;">In case there are no ancestors available, return an empty list.</span></span></span></li>
</ul>
<p><span style="font-size: 18px;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>root[] = [1, 2, 3, 4, 5, 6, 8, 7, N, N, N, N, N, N], target = 7<br></span><span style="font-size: 18px;"><strong><img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/926874/Web/Other/blobid0_1777096989.png" width="457" height="272"> <br>Output: </strong>[4 2 1]<br></span><span style="font-size: 18px;"><strong>Explanation: </strong>The given target is 7, if we go above the level of node 7, then we find 4, 2 and 1. Hence the ancestors of node 7 are 4 2 and 1<br></span></pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>root[] = [1, 2, 3], target = 1<strong>
Output: </strong>[]<br><strong>Explanation: </strong>Since 1 is the root node, there would be no ancestors. Hence we return an empty list.</span></pre>
